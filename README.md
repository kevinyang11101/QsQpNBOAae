# 超市管理系统 java1111

## 项目概述

超市管理系统是一款集成了用户管理、销售管理、人事管理、库存管理和系统管理等功能的信息管理系统。该系统采用了当前流行的前后端分离技术架构，前端使用 Vue.js，后端基于 Spring Boot，确保了系统的高效运行和可维护性。

## 技术栈

- 主要编程语言：Java
- 后端框架：Spring Boot
- 数据持久层框架：MyBatis-Plus
- 缓存技术：Redis
- 前端框架：Vue.js
- 数据库：MySQL

## 运行环境

- 开发环境：IDEA、JDK1.8、Maven、Node.js

## 功能模块

### 用户模块
- 用户登录、注册、权限分配

### 销售管理
- 销售主页
- 购物结账
- 积分兑换
- 销售记录
- 兑换记录

### 人事管理
- 部门管理
- 员工管理

### 库存管理
- 仓库管理
- 出库明细
- 入库明细
- 供货商信息
- 库存统计

### 会员管理
- 会员信息管理

### 系统管理
- 角色管理
- 菜单管理

### 商品管理
- 分类管理
- 商品信息管理
- 积分商品管理
- 销售统计

## 部署步骤

1. 环境准备：确保 JDK、Maven、Node.js 等环境已经正确安装。
2. 数据库搭建：根据提供的数据库脚本初始化数据库。
3. 后端编译：使用 Maven 对后端代码进行编译打包。
4. 前端编译：在项目前端目录下使用 npm 或 yarn 进行编译打包。
5. 部署应用：将编译后的前端静态文件放置到后端指定的 webapp 目录下，然后部署应用到服务器。

## 核心亮点

- 基于微服务的架构设计，提供了高度模块化的系统，易于扩展和维护。
- 前后端分离的架构，使得前端和后端开发可以独立进行，提高了开发效率。
- 引入 Redis 缓存，提升了系统性能，减少了数据库压力。
- 系统权限管理灵活，可以满足不同角色的权限控制需求。
